## 📌 About

This repository contains all frontend projects I am completing as part of the **roadmap.sh frontend development path**. Each project is built from scratch using only the fundamentals — no frameworks, no shortcuts.

The goal is simple: **learn by doing, ship real projects, grow every day.**
